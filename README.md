# Library Management System

## Overview
The **Library Management System (LMS)** is a robust and user-friendly web application designed to streamline and automate day-to-day library operations. The system provides a centralized platform for managing books, members, and transactions, ensuring accuracy, efficiency, and ease of access for both administrators and users.

---

## Objectives
- Automate traditional library processes
- Maintain accurate and up-to-date book records
- Simplify book issue and return workflows
- Minimize manual errors and paperwork
- Improve overall operational efficiency

---

## User Roles
### Admin / Librarian
- Add, update, and remove book records
- Manage registered members
- Issue and return books
- Monitor book availability and transactions

### User / Student
- Browse and search available books
- View issued book details
- Track return dates and status

## Installation and Setup
1. Clone the repository
```bash
git clone <repository_url>
```

2. Navigate to the project directory
```bash
cd Library_Management_System
```

3. Create a virtual environment
```bash
python -m venv venv
```

4. Activate the virtual environment
```bash
venv\Scripts\activate
```

5. Install dependencies
```bash
pip install -r requirements.txt
```

6. Apply migrations
```bash
python manage.py makemigrations
python manage.py migrate
```

7. Run the server
```bash
python manage.py runserver
```
